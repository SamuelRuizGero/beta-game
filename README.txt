How to download the repo?

Steps if you're using Windows:
1. Open your cmd and write the following
2. git clone https://github.com/SamuelRuizGero/beta-game.git
3. Open Netbeans, click "File" -> "Open Project" and then find the folder were you just downloaded the repo
NOTE: You can find where you downloaded the repo by going to your cmd and typing "start ." this will open the
folder that contians the files you downloaded from Github.
